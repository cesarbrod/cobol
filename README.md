# COBOL
Sample COBOL programs for educational purposes only.

Visit the subfolders in this repo to check for the programs.

You will need gnucobol in order to compile and run the programs here.
Usually, on a Debian based box, all you need to do is

`sudo apt install gnucobol`

For further info on GnuCOBOL visit https://gnucobol.sourceforge.io/
